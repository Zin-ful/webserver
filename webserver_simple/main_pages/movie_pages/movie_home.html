WARNING: ADDED MEDIA MAY NOT CONTAIN: SPACES, SPECIAL CHARACTERS. IT IS RECCOMENDED TO ENCODE TITLES IN Base64 IF SUPPORTED BY SERVER VERSION

When adding media to storage, keep in mind the limits of your storage medium. Ill leave some reccomended stats below.

Micro-SD card: Size - 64Gb Minimum, Read - 100Mb/s, Write - any speed;

SSD: Size - 128Gb Minimum, Read - 500Mb/s, Write - any speed;

HDD: Size - 128Gb Minimum, Read - 250Mb/s, Write - any speed;

These sizes are based of the expected amount of content you may be storing per medium.
